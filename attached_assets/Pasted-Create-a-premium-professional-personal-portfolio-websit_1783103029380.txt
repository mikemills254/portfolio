Create a premium, professional personal portfolio website for an experienced Full Stack and AI Software Engineer. The design should NOT look like a traditional developer portfolio with skill bars, technology percentages, animated backgrounds, or flashy effects.

The website should feel like the personal website of a senior consultant or independent software engineer that businesses and startups would hire. Take inspiration from modern personal consulting websites such as Brendan's portfolio style: minimalist, elegant, spacious, and highly professional.

DESIGN REQUIREMENTS:

* Use a LIGHT theme, not dark mode.
* Use lots of whitespace and a premium editorial layout.
* Typography should be large, modern, and confident.
* Use a clean sans-serif font such as Inter, Geist, or SF Pro.
* The design should feel luxurious, trustworthy, and expensive.
* Avoid gradients, neon colors, glassmorphism, excessive animations, or trendy developer portfolio effects.
* Use subtle shadows, soft borders, and a refined color palette.
* The website should look like it was designed by a professional UI/UX agency.

LAYOUT:

* Sticky minimal navigation bar.
* Large hero section with professional portrait/photo placement.
* Strong personal statement and professional introduction.
* Call-to-action buttons such as "View Experience" and "Get In Touch".
* Experience section highlighting years of experience, projects completed, and industries worked in.
* Selected work/case studies section with large elegant cards.
* Services/expertise section.
* Professional timeline or career experience section.
* Testimonials section.
* Contact section with email, LinkedIn, GitHub, and contact form.
* Minimal footer.

CONTENT POSITIONING:
Present me as an experienced software engineer and consultant specializing in:

* Full Stack Development
* Enterprise Software
* AI Applications
* Retrieval-Augmented Generation (RAG)
* Cloud Architecture
* Technical Leadership

Do not focus on listing technologies. Instead, focus on experience, business outcomes, and professionalism.

VISUAL STYLE:

* Apple website level polish.
* Linear and Stripe quality spacing and typography.
* Editorial magazine layout principles.
* Large typography hierarchy.
* Clean cards with subtle shadows.
* Rounded corners used sparingly.
* Premium photography treatment.

TECHNICAL REQUIREMENTS:

* Build using React, TypeScript, and Tailwind CSS.
* Fully responsive design.
* Smooth but subtle animations only.
* Accessibility compliant.
* Production-ready code structure.

The final result should make a potential client immediately think:
"This is an experienced professional that I can trust with my business."
